# Social Network Web Application

A full-stack social media web application built with the Django framework. This project demonstrates backend development, user interaction, and database management capabilities.

## Features
* **User Authentication:** Secure user registration, login, and logout functionality.
* **Post Management:** Users can create, update, and delete their own text posts.
* **Responsive Interface:** Clean and adaptive frontend styled with modern CSS.

## Tech Stack
* **Backend:** Python, Django
* **Frontend:** HTML5, CSS3, JavaScript
* **Database:** SQLite
